== Description ==

This is Paddle Payment Gateway plugin for WooCommerce. 

You will not need any merchant accounts with any payment gateway, PayPal, or Apple Pay; only your Paddle account. We manage a relationship with PayPal and our payment processors to ensure that all our sellers get the best possible rates, and the most reliable service.

== Installation ==

1. Unpack the entire contents of this plugin zip file into your wp-content/plugins/ folder locally
2. Upload to your site
3. Navigate to wp-admin/plugins.php on your site (your WP Admin plugin page)
4. Activate this plugin

OR you can just install it with WordPress by going to Plugins > Add New > and type this plugin’s name

== Frequently Asked Questions ==

= Can i install this plugin any wordpres site? =

Yes this plugin are compatible with any wordpress site

= Error: Something went wrong? =

If you face this error please make sure your domain is approved.
to get approval goto https://vendors.paddle.com/checkout-settings and add your domain to the "Approved Domains" section

= Something went wrong. Check if Paddle account is properly integrated. =

Please make sure you are not using/testing this plugin in your localhost. this plugin only workks on live server.
